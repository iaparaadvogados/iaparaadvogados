# Prompt para Análise de Contratos com IA

Use este prompt para revisar contratos, cláusulas e documentos negociais.

```txt
Atue como assistente de revisão contratual.

Analise o contrato ou cláusula abaixo com foco em clareza, riscos, obrigações, omissões e pontos de atenção.

Contrato ou cláusula:
[cole aqui]

Regras:
- Não dê aconselhamento jurídico definitivo.
- Aponte riscos de forma objetiva.
- Identifique cláusulas ambíguas.
- Destaque obrigações das partes.
- Sugira perguntas que o advogado deve fazer ao cliente.
- Indique pontos que exigem revisão profissional.

Formato da resposta:

1. Resumo do documento

2. Obrigações principais de cada parte

3. Pontos de risco

4. Cláusulas ambíguas ou incompletas

5. Pontos que podem gerar disputa

6. Sugestões de melhoria de redação

7. Perguntas para o cliente

8. Conclusão preliminar
```
