# Prompt para Marketing Jurídico com IA

Use este prompt para criar conteúdos jurídicos educativos, claros e estratégicos.

```txt
Atue como especialista em marketing jurídico, conteúdo educativo e SEO.

Crie um conteúdo sobre o tema abaixo para um público que busca informações jurídicas de forma clara e confiável.

Tema:
[insira o tema]

Público-alvo:
[exemplo: advogados, empresários, consumidores, trabalhadores, aposentados, escritórios de advocacia]

Objetivo:
[atrair tráfego orgânico, educar o público, gerar autoridade, criar post de blog, criar post para LinkedIn etc.]

Regras:
- Use linguagem clara.
- Não prometa resultado.
- Evite tom sensacionalista.
- Não faça captação indevida.
- Mantenha caráter informativo.
- Inclua dúvidas comuns do público.
- Sugira palavras-chave relacionadas.

Formato da resposta:

1. Título principal

2. Meta description

3. Introdução

4. Estrutura sugerida com subtítulos

5. Conteúdo completo

6. FAQ

7. CTA informativo

8. Palavras-chave relacionadas
```
