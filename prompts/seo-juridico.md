# Prompt para SEO Jurídico

Use este prompt para planejar artigos, clusters e conteúdos otimizados para mecanismos de busca.

```txt
Atue como especialista em SEO, conteúdo jurídico e estratégia editorial.

Quero criar um conteúdo otimizado para o tema abaixo:

Tema:
[insira o tema]

Público:
[insira o público]

Objetivo:
[atrair tráfego, gerar autoridade, captar leads, apoiar uma estratégia de cluster etc.]

Tarefa:
Crie uma estratégia de conteúdo SEO para esse tema.

Inclua:

1. Palavra-chave principal

2. Palavras-chave secundárias

3. Intenção de busca

4. Título SEO

5. Meta description

6. H1 recomendado

7. Estrutura de H2 e H3

8. Perguntas frequentes

9. Sugestões de links internos

10. Sugestões de links externos confiáveis

11. CTA recomendado

12. Ideias de conteúdos satélites

Regras:
- Priorize clareza e intenção de busca.
- Evite excesso de repetição de palavra-chave.
- Pense em autoridade temática.
- Considere o público jurídico e de negócios.
```
